# Alexey Mikhail Johll — Computational Science Portfolio

**Singapore · 2025–2026**

Seven interactive experiments in computational science, built as a university and scholarship portfolio. Each experiment begins with a hypothesis Alexey writes himself, followed by an interactive simulation, and concludes with a Socratic dialogue with an AI coach.

The computation amplifies his thinking — it does not replace it.

## How It Works

Each experiment follows the scientific method:

1. **Hypothesis** — Alexey writes his prediction in his own words, then locks it (timestamped, read-only)
2. **The Lab** — Interactive simulation with real-time controls and visualisation
3. **Results** — Summary statistics and key findings from the simulation
4. **Findings** — Alexey interprets the results, then an AI coach asks one Socratic question

## Technical Approach

- Every experiment is a single self-contained HTML file
- No build tools, no frameworks, no package.json
- Pure HTML + CSS + JS with CDN libraries (Chart.js, p5.js, D3.js)
- Works when opened directly in a browser or via GitHub Pages
- All state stored in localStorage — no backend required
