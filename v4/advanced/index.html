Advanced Explorations
Working with research code

These two portals wrap reference Fortran 90 research code for Silicon-Germanium surface science, kept in FortranReference/ at the repo root. I did not originate the algorithms; I ported the physics to run in the browser (via Web Workers) and built instrumentation, sweeps, and 3D visualisations around the core simulations so I could experiment with parameters and understand the results.

These sit outside the main hypothesis-lock workflow of the studio experiments because they are learning tools, not original investigations. If a reviewer wants to know what original scientific thinking I've done, the public exhibit is the honest answer — the three experiments in Football, Fractal, and Random Walk.
