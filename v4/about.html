About
Alexey Mikhail Johll · Singapore · 2025–2026

I'm a secondary school student in Singapore. My interests are football, astronomy, mathematics, and the ways complicated systems are made of simple rules. This site is a working portfolio for university and scholarship applications.

How I actually use this site

Each studio experiment forces me into the same sequence. Before the simulation unlocks, I write my hypothesis in my own words and lock it — the text becomes read-only and timestamped. Then I run the simulation, read the results, and write my interpretation. Only after I've written the interpretation does the coach appear, and it only ever asks one Socratic question at a time. I can reply; it asks another. Nothing about the work gets assigned to me by the computer.

When I finish an experiment I click Export to portfolio. That downloads a JSON file containing my hypothesis, interpretation, coach transcript, logbook, and a snapshot of the key chart. I commit that file into v4/work/. The exhibit page then reads those files and presents the work to anyone visiting the URL. This is why the exhibit is the public evidence and the studio is my private workbench.

What is mine, and what isn't

The three studio experiments — Football, Fractal, Random Walk — are mine. I wrote the hypotheses, interpreted the results, replied to the coach, and made the editorial calls about what to investigate.

The two advanced portals — KMC and MD — are different. They wrap reference Fortran 90 research code for Silicon-Germanium surface science. I did not originate the algorithms. What I did was port the physics to the browser via Web Workers and build instrumentation — visualisations, parameter sweeps, analysis dashboards — around the core simulations so I could experiment with them and see what the parameters do. They're learning work, and the pages say so.

Technical approach

Pure HTML, CSS, and JavaScript. No build tools, no frameworks, no package.json. Libraries (Chart.js, p5.js, Three.js) load from CDNs. State is localStorage plus committed JSON in v4/work/. The AI coach runs through a Cloudflare Worker proxy — if it's unreachable, the studio falls back to a copy-paste-to-Claude workflow so I'm never blocked.

A note on the coach

The coach is set up so it physically cannot give me answers. Its system prompt requires exactly one question per response, no preamble, no praise, no explanation. If I get stuck, it points me at my own words and my own data. I can print out every coach question it ever asked me; I cannot print out an answer it gave me, because it never gives any.
