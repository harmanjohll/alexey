About this portfolio

Every experiment begins with a question I wrote myself. I lock my hypothesis before the simulation runs. I write my own interpretation. The coach asks questions — it never gives answers. When I finish, I export the whole record as a JSON file and commit it, so the exhibit page is my actual work, not an empty promise.
