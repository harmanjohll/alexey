Digital Input — Reading a Button
Outputs are half the story. Now learn to read the world — detect button presses and make decisions with if/else logic.
LESSONThe Concepts

Reading Digital Input

digitalRead(pin) returns either HIGH or LOW — is voltage present or not?

INPUT_PULLUP

Arduino has built-in pull-up resistors. With INPUT_PULLUP, the pin reads HIGH normally and LOW when the button is pressed. This is called active-low logic.

pinMode(2, INPUT_PULLUP); // Pin 2: HIGH when open, LOW when pressed

Making Decisions with if/else

int buttonState = digitalRead(2);

if (buttonState == LOW) {    // Button pressed
  digitalWrite(13, HIGH);    // LED on
} else {                     // Button released
  digitalWrite(13, LOW);     // LED off
}

Your Task

Wire a button to pin 2 and LED to pin 13. When the button is pressed, turn the LED on. When released, turn it off. Click the button in the simulator to test.

Active-low: With INPUT_PULLUP, pressed = LOW, not HIGH. This catches beginners — remember it!
