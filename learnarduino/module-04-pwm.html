PWM & AnalogWrite — Fading an LED
Digital pins are on or off — but PWM fakes analog output by switching on and off thousands of times per second. The result: smooth brightness control.
LESSONThe Concepts

What Is PWM?

PWM (Pulse Width Modulation) rapidly switches a pin between HIGH and LOW. The duty cycle — the percentage of time spent HIGH — controls effective brightness or speed.

  • 0% duty cycle = always OFF (0V effective)
  • 50% duty cycle = half brightness (2.5V effective)
  • 100% duty cycle = always ON (5V effective)

analogWrite()

analogWrite(9, 127);  // 50% brightness (0-255 range)
analogWrite(9, 0);    // OFF
analogWrite(9, 255);  // Full brightness

Range: 0 (off) to 255 (full on). Only works on PWM pins: 3, 5, 6, 9, 10, 11 (marked with ~ on the board).

Fading with a for Loop

// Fade up
for (int brightness = 0; brightness <= 255; brightness += 5) {
  analogWrite(9, brightness);
  delay(30);
}
// Fade down
for (int brightness = 255; brightness >= 0; brightness -= 5) {
  analogWrite(9, brightness);
  delay(30);
}

Your Task

Fade an LED smoothly from off to full brightness and back. Watch the LED's glow change in the simulator.

Not true analog: analogWrite doesn't output a true analog voltage. It's still digital — just switching very fast (490Hz or 980Hz).
