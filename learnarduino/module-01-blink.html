LESSON The Concepts

Every Arduino Sketch Has Two Functions

void setup() {
  // Runs ONCE when board powers on
}

void loop() {
  // Runs FOREVER, over and over
}

setup() is your preparation step — configure pins, set initial states. Runs exactly once.

loop() is where the action happens. After setup finishes, loop runs continuously forever.

Digital Output

Arduino pins have two states: HIGH (5 volts, on) and LOW (0 volts, off).

Before using a pin for output, configure it:

pinMode(13, OUTPUT);    // Pin 13 = output
digitalWrite(13, HIGH); // Turn ON (5V)
digitalWrite(13, LOW);  // Turn OFF (0V)

The delay() Function

  • delay(1000) → pauses 1 second
  • delay(500) → pauses half a second
  • delay(100) → pauses 0.1 seconds
Pin 13 is special — most Arduino boards have a built-in LED on it. LED_BUILTIN is the same as 13.

Your Task

Write a sketch that blinks the LED on pin 13. In setup(), set pin 13 as OUTPUT. In loop(), turn it on, wait 500ms, turn it off, wait 500ms. This creates a 1Hz blink.

Key Concepts

  • Digital signal: only HIGH or LOW. No in-between.
  • Hz (Hertz): cycles per second. 1Hz = one on/off per second.
  • Duty cycle: fraction of time the signal is HIGH. 50% = equal on/off.
