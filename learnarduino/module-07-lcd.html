LCD Display — Showing Text
A 16x2 LCD lets your Arduino talk to the world with text. Learn to initialise the display, position the cursor, and print dynamic data like counters and sensor readings.
LESSONThe Concepts

What Is a 16x2 LCD?

A 16x2 LCD has 2 rows of 16 characters each — 32 characters total. It uses the I2C protocol, which means it only needs 2 data wires (SDA and SCL) plus power and ground.

The LiquidCrystal_I2C Library

#include <LiquidCrystal_I2C.h>

// Address 0x27, 16 columns, 2 rows
LiquidCrystal_I2C lcd(0x27, 16, 2);

void setup() {
  lcd.begin(16, 2);  // Init 16 cols, 2 rows
  lcd.backlight();   // Turn on backlight
  lcd.print("Hello!");
}

Cursor Positioning

Use lcd.setCursor(col, row) to move the cursor. Columns go from 0–15, rows are 0 (top) or 1 (bottom).

lcd.setCursor(0, 0);  // Top-left
lcd.print("Line 1");
lcd.setCursor(0, 1);  // Bottom-left
lcd.print("Line 2");

Useful Functions

  • lcd.clear() — Clears the entire display
  • lcd.print(value) — Prints text or numbers
  • lcd.setCursor(col, row) — Moves cursor position
  • lcd.backlight() / lcd.noBacklight()

Your Task

Display "Hello World" on the first line and a counter that increments every second on the second line. Watch the LCD update in the simulator.

I2C address: Most I2C LCD modules use address 0x27 or 0x3F. Check your module's documentation.
