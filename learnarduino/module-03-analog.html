Analog Input — Reading a Potentiometer
Digital is binary — on or off. Analog reads a range. Learn to read continuous values from a potentiometer and print them to the serial monitor.
LESSONThe Concepts

Analog vs Digital

Digital signals are binary: HIGH or LOW. Analog signals have a range of values — like a dimmer switch versus an on/off switch.

analogRead()

The Arduino's ADC (Analog-to-Digital Converter) reads a voltage from 0V to 5V and converts it to a number from 0 to 1023 — that's 10-bit resolution (2¹⁰ = 1024 values).

int value = analogRead(A0);  // 0 to 1023
// A0 pin at 0V → returns 0
// A0 pin at 5V → returns 1023
// A0 pin at 2.5V → returns ~512

Serial Output

Use the serial monitor to see your readings:

Serial.begin(9600);        // In setup()
Serial.println(value);     // In loop() — prints value + newline
Serial.print("Value: ");   // Prints without newline

Your Task

Read the potentiometer on pin A0 and print the value to the serial monitor. Use the slider below the simulator to change the pot value.

10-bit ADC: 0 to 1023 gives 1024 possible values. Each step = 5V/1024 ≈ 4.9mV. That's the smallest voltage change the Arduino can detect.
