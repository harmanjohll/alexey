About v3

v3 portals use a multi-file architecture: separate Web Workers for computation, modular chart and data management, and external stylesheets. The physics is ported with high fidelity from FORTRAN reference implementations and validated against v2 single-file prototypes. Each portal runs entirely in the browser with no backend.
